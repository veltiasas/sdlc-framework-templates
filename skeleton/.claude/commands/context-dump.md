# Context dump — Guardar estado antes de /clear

Guardá un resumen de esta sesión en docs/_session-notes/session-[NUMERO].md con:

1. DECISIONES TOMADAS: Lista con justificación breve por cada una.
2. PREGUNTAS ABIERTAS: Lo que quedó sin resolver y dónde retomar.
3. PREPARAR PARA SIGUIENTE SESIÓN: Qué traer o pensar antes de la próxima.
4. ARTEFACTOS GENERADOS: Lista de archivos creados o modificados con ruta exacta.
5. ESTADO DEL TRACKER: Qué items se marcaron como completados en esta sesión.

Después indicame que ejecute /clear para limpiar el contexto.
